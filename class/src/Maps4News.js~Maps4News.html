    public request(url: string, method: string, data: string | object, headers: object, bundleResponse: boolean): Promise source

    Request an url using the API token (if available)

    Params:

    NameTypeAttributeDescription
    url string

    Relative or absolute url, api version will be prepended to relative urls
    method string

    Http method
    data string | object

    Raw string or object. If an object is passed it will be encoded and the content-type will be set to application/json
    headers object

    Any headers that should be set for the request
    bundleResponse boolean

    When set to true the promise will resolve with an object {response: Response, data: *}

    Return:

    Promise

    Resolves with either an object, blob, buffer or the raw data by checking the Content-Type header and rejects with ApiError
