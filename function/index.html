    public getPaginatedRange(page: Promise<PaginatedResourceListing> | PaginatedResourceListing, start: Number, stop: Number): Promise<Array<ResourceBase>> source 

    import {getPaginatedRange} from '@mapcreator/maps4news/src/utils/helpers.js'

    Get all the pages from a PaginatedResourceListing or a range

    Params:

    NameTypeAttributeDescription
    page Promise<PaginatedResourceListing> | PaginatedResourceListing

    Promise that returns a PaginatedResourceWrapper
    start Number
    • optional
    • default: 1
    • nullable: true

    Start page
    stop Number
    • optional
    • nullable: true

    Stop page

    Return:

    Promise<Array<ResourceBase>>

    Resolves with an Array containing PaginatedResourceListing instance and rejects with ApiError

    Example:

    import { helpers } from "@mapcreator/maps4news";

const promise = api.users.list(1, 50); // 50 per page is more efficient

helpers.getPaginatedRange(promise).then(data => {
   data.map(row => `[${row.id}] ${row.name}`) // We just want the names
       .forEach(console.log) // Log the names and ids of every user
})

    import {paginateResource} from '@mapcreator/maps4news/src/utils/helpers.js'

    Manually paginate a resource

    Params:

    NameTypeAttributeDescription
    api Maps4News

    Api instance
    url String
    • nullable: true

    Target url, if null it will guess
    page Number
    • optional
    • default: 1

    Page number
    perPage Number
    • optional
    • default: api.defaults.perPage

    Amount of items per page
    Target ResourceBase
    • optional
    • default: CrudBase

    Target class, should extend ResourceBase

    Return:

    Promise

    Resolves with PaginatedResourceListing instance and rejects with ApiError

    Example:

    import { helpers } from "@mapcreator/maps4news";

helpers.paginateResource(api, '/some/custom/endpoint')
